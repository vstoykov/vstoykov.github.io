Latest news

Upgrade to Bootstrap 5

Upgrade from Bootstrap v4 to v5. There is not a big difference visually but to be up to date. Also the jQuery is updated to latest version which forced me finally to use btter plugin for the gallery. Also few more touches on the gallery to behave more like real one. Someday I need to spend time and select more images eventually. For now it is what it is.

Posted on Jan 18, 2026 by Venelin Stoykov

Migrate to using Bootstrap

The new version of this site is using bootstrap now instead of custom styles and jQueryUI. It's havier than my old custom styles, but it looks a lot more pretty and modern on all devices. Currently it uses bootstrap CDN for both styles and javascript. In future I will migrate styles to custom build from SCSS with the features that are needed for this theme.

Posted on July 8, 2018 by Venelin Stoykov
